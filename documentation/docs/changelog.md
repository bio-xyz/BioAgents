## Version History

### [0.1.0] - Coming Soon

#### Added
- Initial release of BioAgents framework
- Agent Framework with tools system
- Deep Research capabilities
- LLM library with multiple provider support
- x402 payment integration
- Vector database and embeddings support
- Character system for agent customization
- Client UI with React components
- Docker deployment support

#### Tools
- Planning Tool
- Knowledge Tool
- Hypothesis Tool
- File Upload Tool
- Semantic Scholar integration
- Reply Tool

#### Features
- Multi-turn conversations
- State management
- Document processing
- Embedded wallet support
- Gasless USDC payments
